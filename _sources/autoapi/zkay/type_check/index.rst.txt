:mod:`zkay.type_check`
======================

.. py:module:: zkay.type_check

.. autoapi-nested-parse::

   This package contains modules which provide functionality related to type-checking zkay code.

   ==========
   Submodules
   ==========
   * :py:mod:`.type_checker`: Zkay type checker implementation
   * :py:mod:`.type_exceptions`: Exceptions raised by the type checker
   * :py:mod:`.contains_private`: Visitor to check whether a subtree contains any private expressions
   * :py:mod:`.final_checker`: Visitor to check whether the AST obeys the rules for final variables.
