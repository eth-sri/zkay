Module Contents¶

zkay.utils.run_command.run_command(cmd: List[str], cwd=None, allow_verbose: bool = False) → Tuple[Optional[str], Optional[str]]¶

Run arbitrary command.

Parameters

  • cmd – the command to run (list of command and arguments)

  • cwd – if specified, use this path as working directory (otherwise current working directory is used)

  • allow_verbose – if true, redirect command output to stdout (WARNING, causes return values to be None)

Returns

command output and error output (if not (allow_verbose and cfg.verbosity))
