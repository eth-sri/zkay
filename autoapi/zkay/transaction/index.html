zkay.transaction¶

This package contains the runtime API used by the offchain simulation code when working with zkay transactions.

Submodules¶

  • interface: Runtime API interface

  • offchain: Offchain simulator base class with common functionality

  • runtime: Static class which provides access to the individual API backend singletons.

  • types: Type wrapper classes (for safer API interactions) used by the Runtime API.

Subpackages¶

  • blockchain: Blockchain backends

  • crypto: Cipher backends

  • keystore: Keystore backends

  • prover: Proof-generation backends
