zkay.zkay_ast.pointers¶

This package contains modules which resolve references to other AST elements.

Submodules¶

  • parent_setter: Visitor which sets the parent, statement and function references for each element to the correct values.

  • pointer_exceptions: Exceptions raised within this module

  • symbol_table: Construct symbol table from AST and resolve identifier and user-defined-type targets.
