zkay.zkay_ast.visitor

This package contains several basic AST visitors.

Submodules

  • deep_copy: Fast custom deep copy implementation for AST

  • function_visitor: Only visits contract functions (mode node-or-children).

  • python_visitor: Returns python code corresponding to AST

  • solidity_visitor: Returns solidity code corresponding to AST

  • statement_counter: Returns number of statements in AST

  • transformer_visitor: Visitor base class which replaces visited AST elements by the visit function’s return value.

  • visitor: AST visitor base class
