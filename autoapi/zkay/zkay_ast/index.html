zkay.zkay_ast

This package contains modules with functionality related to building, traversing, transforming and outputting ASTs.

Submodules

  • ast.py: Defines all AST elements

  • build_ast.py: Parser wrapper, which manually constructs AST elements for which there is no 1:1 correspondence to the antlr parser output.

  • global_defs.py: Definitions for globally accessible solidity builtin functions and variables.

  • process_ast.py: Takes a raw AST and performs pre-processing and type-checking.

Subpackages

  • analysis: Program analysis functionality

  • pointers: Functionality for resolving references to other AST elements.

  • visitor: Visitor classes for processing ASTs
