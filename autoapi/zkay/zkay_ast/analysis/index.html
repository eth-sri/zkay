zkay.zkay_ast.analysis

This package contains modules implementing static analysis.

Submodules

  • alias_analysis: Alias analysis to determine whether the privacy labels at different locations refer to the same address.

  • call_graph: Compute sets of transitively called functions for each function.

  • circuit_compatibility_checker: Determine whether the private parts of an AST can be expressed using proof circuits.

  • contains_private_checker: Determine whether element contains any private expressions.

  • hybrid_function_detector: Determine which functions require verification.

  • loop_checker: Ensure that loops do not contain private expressions.

  • partition_state: Helper class to store alias analysis state.

  • return_checker: Ensure that there is at most one return statement per function at the end of the body.

  • side_effects: Determine whether element contains side effects.
