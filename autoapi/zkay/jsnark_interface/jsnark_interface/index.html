Compile the given circuit java code and then compile the circuit which it describes using jsnark.

Parameters

  • circuit_dir – output directory

  • javacode – circuit code (java class which uses the custom jsnark wrapper API)

Raises

SubprocessError – if compilation fails

zkay.jsnark_interface.jsnark_interface.prepare_proof(circuit_dir: str, output_dir: str, serialized_args: List[int])¶

Generate a libsnark circuit input file by evaluating the circuit in jsnark using the provided input values.

Parameters

  • circuit_dir – directory where the compiled circuit is located

  • output_dir – directory, where to store the jsnark output files

  • serialized_args – public inputs, public outputs and private inputs in the order in which they are defined in the circuit

Raises

SubprocessError – if circuit evaluation fails
