zkay.jsnark_interface¶

This package contains glue code to interact with the external jsnark and libsnark interfaces.

Submodules¶

  • jsnark_interface: Jsnark circuit compilation and evaluation (preparation steps for key and proof generation).

  • libsnark_interface: Libsnark key and proof generation.
