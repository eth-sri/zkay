zkay.compiler.privacy.proving_scheme.backends.groth16¶

This module defines the verification key, proof and verification contract format for the Groth16 proving scheme

See “On the Size of Pairing-based Non-interactive Arguments”, Jens Groth, IACR-EUROCRYPT-2016 https://eprint.iacr.org/2016/260
