zkay.compiler.privacy

This package deals with zkay compilation.

Submodules

  • library_contracts: Stores strings which contain pki and library contract solidity code

  • manifest Defines the entries of the zkay manifest file which stores compilation metadata.

  • offchain_compiler Offchain simulation code generator.

Subpackages

  • transformation: Contains modules for transforming zkay ASTs into solidity ASTs + abstract proof circuits

  • circuit_generation: Contains modules for compiling abstract proof circuits into backend specific representations.

  • proving_scheme: Contains modules for generating verification contracts for different proving schemes.
